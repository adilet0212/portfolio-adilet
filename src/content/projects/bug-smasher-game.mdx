## Overview
**Bug Smasher** is a browser game designed to test reaction time and precision.  
Players must click on a bug that moves faster after each successful hit with difficulty scaling to keep it engaging.  

The game was developed as part of a **client-side programming project** and showcases interactive DOM manipulation and event-driven gameplay logic.

<Figure src="/images/projects/bug-smasher-game/cover.png" caption="This is how game looks like" gif />

## Features
- Dynamic speed adjustment after every successful click.
- Score tracking and simple level progression system.
- CSS-based animations for bug motion and effects.
- Fully playable on desktop and mobile browsers.
